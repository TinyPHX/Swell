Detailed Description

Dynamic mesh used to render water surface.

Simple Mesh

A mesh can be very simple, using only SwellMesh.StartGridSize and SwellMesh.MaxSize to form the grid.

Advanced Mesh

Alternatively you can use levels to define a mesh that gets progressively less coarse as it gets further from it's center.
