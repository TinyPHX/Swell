Returns exact wave height of the water mesh at some x, z value.

This will check first to see if the position height has been calculated this frame, if not it will calculate it as long as calculate is set to true. li is the level of fidelity used in the SwellMesh.

Parameters
xPositionx position on the mesh
yPositiony position on the mesh (z position in unity global space)
calculateIf false this will only return heights that have already been calculated this frame.
liThe grid level (fidelity) used to get the position from

◆ GetWaterHeight()

float Swell.SwellWater.GetWaterHeight ( Vector3  position)
inline

Gets the exact height of the SwellMesh.

Returns y intersection value of the water mesh at some x, z value. To get the exact position of the mesh we get the height of the nearest vectors and then interpolate between them.

Parameters
positionThe position (x, z) at which to check the water height.

Example

Returns the height of the water at the origin of the scene:

GetWaterHeight(Vector3.Zero)
